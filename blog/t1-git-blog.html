Version control is very important because it allows you to revert to a stable build in case you mess up your code. Without version control, users would have to manually revert their code. This would be extremely time consuming. Version control also helps track coding changes. Each time a new version comes out, users can mark the changes they've made through notes. Lastly, version control provides backup space for users. If a person loses a laptop with Git, that person can go through GitHub to head right back in.
