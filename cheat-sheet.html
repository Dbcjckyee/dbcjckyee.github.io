    • RUBY
    Method Example Explanation
    .each_slice (1..8).each_slice(4)= [1,2,3,4][5,6,7,8] Acts off a collection and takes an integer argument. Divides collection into arrays of (integer) length.
    .any? [124, 100, 800].any? {|num| num % 4 == 0} = true Goes through a collection and passes each element through the block. If there is a true element in the collection, it returns true.
    .transpose [["a","b","c","d"],["e","f","g","h"]].transpose = [["a","e"],["b,"f"],["c","g"],["d","h"]] Takes an array an pairs the 1st elements together in a new array. Does the same for each subsequent pair (2nd+2nd, 3rd+3rd, etc.).
    .take_while [1, 2, 3, 4, 5, 0].take_while { |i| i < 4 } = [1,2,3] Returns all elements that fall in the block's conditions until nil/false is returned. Stops returning then.
    SQL
    Keyword Example Explanation
    SELECT select * from countries Returns all items in column [*] within the database of countries.
    INNER JOIN FROM names INNER JOIN people Links two databases together and returns rows with a match that appear in both tables.
    ORDER BY states.id ORDER BY region.id ASC/DESC Takes the first column and sorts the results by the second column. Can order items in ASCending or DESCending fashion.
    BETWEEN WHERE state.id BETWEEN 1 AND 6 Looks at the column after WHERE and returns all matches that fall between the arguments that are within BETWEEN.
    JAVASCRIPT
    Method Example Explanation
    .reduce [1,2,3,4,5].reduce(function(a,b){return a+b;}) = 15 Takes a collection and applies the function in the reduce argument to it. Useful for combining objects and finding totals.
    .filter [4,10,20,30].filter(function(a){return a > 15}) = [20,30] Takes a collection and returns the values that meet the criteria supplied in the argument's function.
    .indexOf ["steve","jam","cat","people"].indexOf("jam") = 1 Returns the index of the supplised argument within the collection it is in.
    .sort [4, 2, 5, 1, 3].sort(function(a,b){return a-b}} = [1,2,3,4,5] Takes a collection and sorts it in ascending or descending order.
